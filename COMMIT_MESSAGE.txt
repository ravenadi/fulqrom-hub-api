- Add resource-specific ID fields to audit log response
- Fix per-module resource restriction logic
- Increase rate limit from 100 to 1000 requests/15min for development

Permission Logic (Per Module):
- If user has role permission + resource_access for module: Show ONLY those resources
- If user has role permission + NO resource_access: Show ALL resources
- If user has NO role permission: Block all activities (even with resource_access)
- Super Admin & Admin: Bypass all filters
